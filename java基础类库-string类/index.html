String、StringBuffer 和 StringBuilder 类

字符串类

字符串就是一连串的字符序列，Java 提供了String 和StringBuffer 两个类来封装字符串，并提供了一系列方法来操作字符串对象。

String 类：

String 类是不可变类，即一旦一个 String 对象被创建以后，包含在这个对象中的字符序列是不可改变的，直至这个对象被销毁。

StringBuffer 类：

StringBuffer 对象则代表一个字符序列可变的字符串，当一个 StringBuffer 被创建以后，通过 StringBuffer 提供的 append()、insert()、 reverse()、 setCharAt()、 setLength() 等方法可以改变这个字符串对象的字符序列。一旦通过 StringBuffer 生成了最终想要的字符串，就可以调用它的 toString() 方法将其转换为一个 String 对象。

StringBuilder 类：

StringBuilder 和 StringBuffer 基本相似；不同的是 StringBuilder 是线程安全的，所以应该优先考虑 StringBuilder 类。

操作字符串的常用方法

  1. char charAt(int index)： 获取字符串中指定位置的字符；0 ~ leng()-1

    String s = new String("abcdefg");
s.charAt(5);
   
运行结果:
e

  2. int compaerTo(String srt)： 比较两个字符串的大小

    • 两个字符串字符序列相等，返回 0；
    • 长度短的与长度长的字符一样时，返回第一个不相等的字符差；
    • 字符不一样时，返回长度差；
    (1)
String a = "abc";
String b = "abc";
返回：num = 0
   
(2)
a="h";
b="hello";
返回:	num=4;
   
(3)
a="assdf";	//'a'=97
b="bdd";	//'b'=98
   
'a'-'b'==-1
返回:	num=-1;

  3. String concat(String str)： 将该 String 对象与 str 连接在一起

  4. int length()： 返回字符串长度

  5. int indexOf()： 找出该字符第一次出现的位置

  6. int lastIndexOf()： 找出该字符最后一次出现的位置

  7. char toCharArray()： 将该 String 对象转换为 char 数组

  8. toUpperCase()、toLowerCase()： 转换为大写、小写

  9. String[] split()： 根据正则表达式的匹配来拆分字符串

    String str = "boo:and:foo";
String[] arr1 = str.split(":");
String[] arr2 = str.split("o");
   
运行结果:
arr1	{ "boo", "and", "foo" }　　
arr2	{ "b", "", ":and:f" }

  10. boolean equals()： 相同字符序列时，返回 true

  11. boolean equalsIgnoreCase()： 忽略大小写比较字符序列

  12. boolean contains()： 判断一个字符串里面是否包含指定的内容，返回一个布尔值

  13. boolean startsWith()、endsWith()： 测试此字符串是否以指定的前缀开始。返回一个布尔值

  14. trim()： 去掉字符串左右空格

  15. String replace(char old, char new)： 字符串替换

  16. String replaceAll()： 正则表达式替换内容

  17. String substring(int bedginIndex, int endIndex)： 截取字符串

    String str = "abcdefg";
str.substring(0, 3);	//[0-3) 位置的内容
str.substring(2);		//[2,?) 位置的内容
    
运行结果:
abc
cdefg

StringBuffer 和 StringBuilder 类

StringBuffer、StringBuilder 有两个属性: length 和capacity。

length 属性表示其包含的字符序列的长度。与String 对象的 length 不同的是，StringBuilder、StringBuffer 的 length 是可以改变的，可以通过 length()、 setLength(int len) 方法来访问和修改其字符序列的长度。

capacity 属性表示 StringBuilder 的容量，capacity 通常比 length大，程序通常无须关心 capacity 属性。

StringBuilder sb = new StringBuilder;

sb.append("java");			追加字符串: sb="java"
sb.insert(0, "hello ");		插入: sb="hello java"
sb.replace(5, 6, ",");		替换: sb="hello, java"
sb.delete(5, 6);			删除: sb="hellojava"
sb.reverse();				反转: sb="avajolleh"

sb.length();				输出: 9
sb.capacity();				输出: 16
sb.setLength(5);			改变长度，将只保留前面部分: sb="avajo"
