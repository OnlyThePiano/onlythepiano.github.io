Java 基础类库-Object类

Object 类

Object 类是 java 默认的提供的一个类，Object 类是所有类的父类，也就是说任何一个类的定义的时候如果没有明确的继承一个父类的话，那么它就是 Object 的子类。

- 常用方法：

  1. Class <?> getClass() 方法

    返回该对象的运行时类；final 方法，返回 Class 类型的对象，反射来获取对象。

  2. String toString() 方法

    一般子类都有覆盖，目的在于获取对象的信息。

  3. protected void finalize() 方法

    该方法用于释放资源。因为无法确定该方法什么时候被调用，很少使用。

  4. boolean equals() 方法

    比较对象的内容是否相等。

  5. int hashCode() 方法

    该方法用于哈希查找，重写了 equals 方法一般都要重写 hashCode 方法。这个方法在一些具有哈希功能的 Collection 中用到。

toString 方法

toString 源码：

在使用对象直接输出的时候，默认输出的是一个对象在堆内存上的地址值；如若要输出该对象的内容，则要覆写toString()方法。

public String toString() {
    return getClass().getName() + "@" + Integer.toHexString(hashCode());
}

equals 方法

equals 源码：

在源码中，传递来的Object对象和当前对象比较地址值，返回布尔值。所以，默认进行比较的是两个对象的地址。

public boolean equals(Object obj) {
    return (this == obj);
}

重写 equals 方法：

判断 Person 类的两个对象的姓名是否相同时，重新覆写equals()

@Override
public boolean equals(Object obj) {
    if (obj == null)
        return false;
    if (this == obj) {
        return true;
    }
    
    //判断对象是否属于同一个类
    if (!(obj instanceof Person)) {
        return false;
    }
    
    //父类转换为子类
    Person stu = (Person)obj;
    return stu.name.equals(this.name) && stu.age == this.age;
}


Person per1 = new Person("1", 18);
Person per2 = new Person("2", 18);
System.out.println(per1.equals(per2));
