Windows、Linux系统目录

  • 在Windows操作系统中，是先将物理地址分开，再在分区上建立目录.在Windows操作系统中，所有路径都是从盘符开始，如C://program file。

  • Linux正好相反，是先有目录，再将物理地址映射到目录中。在Linux操作系统中，所有路径都是从根目录开始。Linux默认可分为3个分区，分别是boot分区、swap分区和根分区。

在Linux操作系统中，用户可根据需要进行修改分区.修改后的分区中，同一目录下的文件可能在不同分区中.比如/home目录下有a、b、c三个目录，然后有三个分区分别挂载到这三个目录下，这种操作是允许的；

Windows的分区是物理上的概念，就像我们把一块硬盘分成C:，D:，E:三个区一样，物理上将存储空间分开；

/ 、 /home、~、目录区别

  • cd / 是跳转到根目录；

  • cd /home 相当于查看有多少普通用户的家目录，有多个用户时，所有的普通用户的父目录都是home目录；

  • cd ~ 是跳转到当前用户的家目录；

这里如果是Linux系统应该是：/home/79949，而Windows的“家目录”好像是/c/Users

由于linux一切都是由根目录开始，所以绝对路径写法是 cd /目录；

cd /a 表示跳转根目录下的a目录；
#
cd a/b 表示跳转a目录下的b目录；不同在于一个是根目录开始，一个是a目录为开始；

可以将/理解为boss目录，一切绝对路径的起点；
