github上有分支master、dev：

在本地：

  • git checkout -b dev 新建并且切换到本地dev分支
  • git pull origin dev 本地分支和远端分支dev相关联
  • 或
  • git branch -u origin/dev 本地分支和远端分支dev相关联

github上没有dev分支：

  • git push origin :

  • git push origin dev:dev
#或
git push origin dev
#相当于默认建立与本地同名
